## How to use this index

- **Operator looking for "what changed in vX.Y"**: start here, click through to CLAUDE.md anchor.
- **Agent reading the project**: read AGENTS.md for cross-CLI invariants, then CLAUDE.md for Claude-Code-specific runtime, then this index if a specific release is referenced.
- **Investigating a regression**: CHANGELOG.md has the full chronological list. This index groups by theme.
